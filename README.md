# Discord Music Bot

A discord music bot using `discord.js` and TypeScript

## Development

Rename `config.example.ts` to `config.ts` then add your Discord bot token.

Run dev server

```bash
pnpm dev
```

Build server

```bash
pnpm build
```

Run server

```bash
pnpm start
```
